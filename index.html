Focus, enforced.

A commitment enforcement tool. Not a productivity system.

What it does

You choose an already-open app — Pages, VS Code, Excel, a browser tab.

No Frills Focus puts a physical tunnel on your screen: your chosen app stays visible, and everything else is blocked by solid, opaque masks.

It doesn’t try to improve you.

It helps you enforce a decision you already made.

How it works
  • Open the app you want to work in.
  • Select it from the picker.
  • Choose a timer (15/30/60+) or “until I’m done.”
  • Exit anytime with keyboard shortcuts (visible on-screen).
EXIT: Cmd + Shift + X QUIT: Cmd + Shift + Z

No launching apps. No file access. No control over your tools. Just a screen tunnel that does what you told it to do.

What it is not
  • Not AI. Not analytics. Not telemetry.
  • Not an account. Not a subscription.
  • Not a browser extension. Not a system monitor.
  • Not a “focus ecosystem.”

Install / First launch
  • Download the ZIP.
  • Unzip it.
  • Drag No Frills Focus.app into /Applications.

If macOS blocks it:

Right-click No Frills Focus.app → Open → Open again.

Or: System Settings → Privacy & Security → “Open Anyway”.

This is a small utility. It runs locally. No accounts. No network dependency. Just a commitment to yourself.

Note: the app may be unsigned. macOS can warn on first launch. That’s normal for indie utilities.
